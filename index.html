Code Generation and Scaffolding

$ npx create-nx-workspace@latest awesome-workspace $ nx g @nrwl/next:app site --directory=apps $ nx g @nrwl/react:component my-component --project=site $ nx g @nrwl/nest:app api --frontendProject=site $ nx g @nrwl/js:lib shared --importPath=@awesome-lib/shared

Demo

Run Tasks

$ nx run --target=build --projects=api $ nx affected --target=test $ nx affected --target=lint --base=master $ nx format:check $ nx format:write

Cache Task Result

{ "cacheableOperations": ["build", "test"] }

Explore the Graph

Visualize your workspace and its dependencies.

                $ nx dep-graph

Distribute Task Execution (DTE)

$ nx run-many --target=build --projects=api,app,som-lib --parallel=3
